# AWS Deadline Cloud for Blender User Guide

This guide provides step-by-step instructions for using AWS Deadline Cloud with Blender to render your projects faster by distributing rendering tasks across multiple machines.

## What's Inside

This user guide covers:

- [Installation](installation/index.md) - How to install the Blender submitter
- [Using the submitter](using-submitter.md) - How to prepare and submit rendering jobs
- [Telemetry](telemetry.md) - How this repository uses telemetry and how to opt out
